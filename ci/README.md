# Continuous Integration

Dieser Ordner sammelt Konfigurationen für automatisierte Tests, Builds und Deployments.

Lege hier beispielsweise GitHub-Actions-Workflows, Jenkins-Pipelines oder andere CI-Konfigurationen ab.
Dokumentiere Abhängigkeiten und Secrets in einer gesicherten Umgebung und verlinke entsprechende Hinweise im Ordner `docs/`.
